# Privacy Policy for SnipBit

Effective date: April 25, 2026

SnipBit is a Chrome extension for saving, organizing, searching, favoriting, copying, and reusing text snippets.

## Data SnipBit stores

SnipBit stores data that users create or explicitly save in the extension, including:

- Text snippets
- Snippet titles
- Snippet collections
- Favorite status
- Extension preferences

If a user uses the right-click “Save selection to SnipBit” action, SnipBit saves the selected webpage text as a snippet.

## How data is stored

SnipBit stores extension data locally in the user’s browser using Chrome extension storage.

SnipBit does not require an account, does not use a backend server, and does not sync user data to a cloud service.

## How data is used

SnipBit uses stored data only to provide its core features:

- Saving snippets
- Searching snippets
- Favoriting snippets
- Copying snippets
- Organizing snippets into collections
- Managing snippets in the popup and side panel

## Data sharing

SnipBit does not sell, rent, transfer, or share user data with third parties.

SnipBit does not use user data for advertising, analytics, tracking, creditworthiness, lending, or unrelated purposes.

## Remote code

SnipBit does not load or execute remote JavaScript, remote WebAssembly, CDN-hosted extension code, or remotely hosted extension logic.

All extension code is included in the extension package submitted to the Chrome Web Store.

## Analytics and tracking

SnipBit does not use analytics, advertising, tracking pixels, behavioral monitoring, or background browsing activity collection.

## Website content

SnipBit only saves webpage content when the user explicitly selects text and chooses the SnipBit right-click save action.

SnipBit does not automatically read, monitor, scrape, or transmit webpage content.

## User control

Users can edit or delete snippets and collections inside the extension.

Users can also remove locally stored SnipBit data by uninstalling the extension or clearing the extension’s stored data in Chrome.
